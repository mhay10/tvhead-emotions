TVead Expression API Reference

Endpoints

API Usage Documentation

Endpoint: /
Method: GET
Description: Returns this API usage documentation

Set Expression

Endpoint: /set-expression
Method: POST
Content Type: application/json
Description: Sets the current expression

Request Body: {"expression": "happy"}

Example Request:

  curl -X POST http://localhost:5000/set-expression \
       -d '{"expression": "happy"}'

Reset Expression

Endpoint: /reset-expression
Method: POST
Description: Resets the current expression to default

Example Request:

  curl -X POST http://localhost:5000/reset-expression
